# Basic XML input format

For the moment, the following format is used:

      <item>
        <title>Item Title</title>
        <geogebra>GeoGebra material ID, e.g. rrfkyqtq</geogebra>

        <statement>
          HTML code of the statement.
        </statement>

        <answer>
          HTML code of the hidden answer.
        </answer>
      </item>
